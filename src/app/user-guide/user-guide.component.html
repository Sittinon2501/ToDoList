คู่มือการใช้งานระบบ Task To-Do List

ระบบ Task Management นี้เป็นเครื่องมือที่ช่วยให้คุณสามารถเพิ่ม จัดการ และจัดลำดับความสำคัญของงานต่างๆ ได้อย่างมีประสิทธิภาพ โดยมีฟีเจอร์ต่างๆ ดังนี้:

1. เพิ่มงานใหม่

ในหน้า Tasks คุณสามารถเพิ่มงานใหม่ได้โดยกรอกข้อมูลในฟอร์มทางซ้ายมือ ซึ่งมีฟิลด์ที่จำเป็นดังนี้:

  • งานที่ต้องทำ - ชื่อของงานที่ต้องการเพิ่ม
  • รายละเอียด - รายละเอียดเพิ่มเติมเกี่ยวกับงาน
  • กำหนดการ - วันที่ครบกำหนดส่งงาน
  • ความสำคัญ - ระดับความสำคัญของงาน (สูง, กลาง, ต่ำ)

เมื่อกรอกข้อมูลครบถ้วนแล้ว ให้กดปุ่ม "เพิ่มงาน" เพื่อบันทึกงานใหม่ลงในระบบ งานจะถูกจัดกลุ่มตามลำดับความสำคัญและแสดงในส่วนขวามือของหน้า Tasks

2. จัดลำดับความสำคัญของงานด้วย Drag and Drop

งานทั้งหมดจะถูกจัดเรียงตามระดับความสำคัญ ได้แก่ สูง (สีแดง), กลาง (สีเหลือง), และ ต่ำ (สีเขียว) คุณสามารถย้ายงานระหว่างกลุ่มความสำคัญได้โดยใช้ฟีเจอร์ Drag and Drop ดังนี้:

  • กดคลิกเมาส์ค้างที่งานที่ต้องการย้าย
  • ลากงานไปยังกลุ่มระดับความสำคัญที่ต้องการ
  • ปล่อยเมาส์เพื่อย้ายงานไปยังตำแหน่งใหม่ ระบบจะบันทึกการเปลี่ยนแปลงให้อัตโนมัติ

3. เปลี่ยนสถานะของงาน

คุณสามารถติดตามสถานะของงานได้โดยการเปลี่ยนสถานะของงานเป็น:

  • Pending - งานที่ยังไม่เริ่มทำ
  • In Progress - งานที่กำลังดำเนินการอยู่
  • Done - งานที่เสร็จสิ้นแล้ว

คุณสามารถเปลี่ยนสถานะได้โดยกดปุ่มที่เหมาะสม (เช่น "เริ่มงาน" หรือ "เสร็จสิ้น") บนงานที่ต้องการ

4. ลบงาน

คุณสามารถลบงานออกจากระบบได้โดยการกดปุ่ม "ยกเลิก" ในงานที่ต้องการลบ งานจะถูกลบออกจากรายการในทันที

5. ดูประวัติงานที่เสร็จสิ้น

ในหน้า History คุณสามารถดูรายการของงานทั้งหมดที่มีสถานะ Done โดยงานจะถูกจัดแยกตามลำดับความสำคัญ (สูง, กลาง, ต่ำ) เพื่อให้คุณสามารถติดตามงานที่เสร็จสิ้นแล้วได้ง่ายขึ้น

6. ระบบจัดการงานตามลำดับความสำคัญ

ในหน้า Home ระบบจะแสดงงานตามลำดับความสำคัญและแสดงเฉพาะงานที่มีสถานะ Pending หรือ In Progress เท่านั้น:

  • ระดับความสำคัญสูง: ระบบจะแสดงงานในระดับความสำคัญสูงก่อน
  • ระดับความสำคัญกลาง: เมื่องานในระดับสูงเสร็จสิ้นทั้งหมดแล้ว ระบบจะแสดงงานในระดับความสำคัญกลางต่อไป
  • ระดับความสำคัญต่ำ: เมื่องานในระดับกลางเสร็จสิ้นทั้งหมด ระบบจะแสดงงานในระดับความสำคัญต่ำต่อไป

หากไม่มีงานที่ตรงกับสถานะที่กำหนดในกลุ่มนั้นๆ ระบบจะแสดงปุ่มให้คุณกดเพื่อไปยังหน้า Tasks เพื่อเพิ่มงานใหม่ได้อย่างสะดวก

หากต้องการความช่วยเหลือเพิ่มเติม คุณสามารถติดต่อทีมผู้พัฒนาได้ที่หน้า เกี่ยวกับผู้พัฒนา
